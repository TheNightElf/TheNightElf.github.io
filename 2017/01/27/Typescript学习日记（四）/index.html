函数重载

不同于 javascript 的函数重载，typescript 中重复的对于一个函数进行定义，只要 参数与返回值类型 不同，就不会相互覆盖。

这里需要注意的是：如果通过重载对函数进行约束之后，那么该函数只能接受重载的定义类型。
